# Keyboard shortcuts

GhostPad follows standard desktop shortcut conventions (via Qt `StandardKey`
where possible, so they adapt to platform norms).

## File

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New tab |
| `Ctrl+O` | Open… |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As… |
| `Ctrl+W` | Close tab |
| `Ctrl+Q` | Quit |

## Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+X` | Cut |
| `Ctrl+C` | Copy |
| `Ctrl+V` | Paste |
| `Ctrl+A` | Select all |
| `Ctrl+D` | Duplicate line |

## Find & replace

| Shortcut | Action |
|----------|--------|
| `Ctrl+F` | Find… |
| `F3` | Find next |
| `Shift+F3` | Find previous |
| `Ctrl+H` | Replace… |
| `Escape` | Close find panel |

## View

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+W` | Toggle word wrap |
